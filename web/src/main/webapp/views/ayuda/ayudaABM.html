En la sección {{ paquete }} podemos visualizar, crear, editar y dar de baja {{ entities }} en el sistema S.A.P.O.

Consultar {{ entities }}

Corresponde a la vista general de la sección {{ paquete }} y desde aquí se podrá acceder al resto de las acciones que pueden ser realizadas sobre {{ entities }}.

El sistema ofrece una pantalla para consultar las {{ entities }} existentes en él. La búsqueda se puede acotar utilizando los campos filtros que aparecen en pantalla. Al presionar el botón de "Buscar" (lupa), se desplegará una lista con los resultados de la búsqueda.

Al posicionarse sobre alguno de los resultados, se mostrarán las opciones "Editar" (lápiz), "Ver" (ojo) y "Dar de baja" (basurero).

Asimismo, en el rincón inferior derecho, se podrá apreciar un botón circular para registrar una nueva entidad.

Dar de baja {{ entity }}

Para dar de baja de una {{ entity }}, seleccione la opción "Dar de baja" (en forma de basurero) que aparecerá cuando se posicione sobre uno de los elementos de la lista. A continuación, se desplegará un diálogo para que complete el motivo de baja.

Finalmente, presione “Aceptar” para confirmar la acción.

Registrar {{ entity }}

Al seleccionar la opción "Registrar nuevo" desde "Consultar {{ entities }}", se desplegará la siguiente pantalla para registrar una nueva entidad.

Seleccione el o los roles del usuario, marcando las casillas de verificación correspondientes. Una vez registrado el usuario, se le enviará un mail al correo electrónico registrado con los datos del mismo.

Para las cátedras, se presentan tres secciones a completar: "General", "Horarios" y "Trabajos prácticos".

General: El sistema nos solicita los datos que identificarán a la cátedra tales como ser nombre, materia y descripción.

Horarios: En esta pantalla podemos especificar los días y horarios en los cuales se dicta la cátedra.

Trabajos prácticos: Permite determinar qué trabajos prácticos se realizarán en el cursado de dicha cátedra. Busque los trabajos prácticos correspondientes a la cátedra y márquelos en la casilla de verificación que aparece a la derecha. A medida que seleccione los trabajos prácticos, los mismos se irán agregando a la cátedra.

Para quitar un trabajo práctico perteneciente a la cátedra, marque como vacía la casilla de verifiación.

Complete los campos y confirme la acción presionando "Aceptar". En caso de no completar los campos obligatorios, los mismos se marcarán en rojo, con un mensaje aclaratorio.

Modificar {{ entity }}

Cuando se desea modificar una {{ entity }} se debe buscarla en la pantalla de consulta. Una vez que se identificado, seleccionamos la opción "Modificar" (símbolo en forma de lápiz) que aparece en el lado derecho del elemento seleccionado.

Luego, el sistema mostrará una misma pantalla similar a la presentada en el registro de una nueva entidad, con los campos de la entidad seleccionda para ser modificada. Finalmente, las correcciones efectuadas se guardarán haciendo un “Click” en el botón “Aceptar”.
